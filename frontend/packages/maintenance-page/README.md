# Maintenance page

Displayed when eVaka sites are down for maintenance.

## Requirements

* Node version >= 10
* Yarn version >= 1.16

## Development

Start development server hosting the site at <http://localhost:1234>

```sh
yarn dev
```

Run linter with optional autofix

```sh
yarn lint[:fix]
```

Build

```sh
yarn build
```
